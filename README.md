juego de tateti

## Juego Aviones

Se ha añadido un juego simple de aviones en `aviones.py`.

Ejecución (PowerShell):

```powershell
python .\\aviones.py
```

Controles:
- Introduce coordenadas como `A5` o `5 1` para disparar.
- Escribe `q` para salir.

### Versión gráfica (Tkinter)

También hay una versión gráfica en `aviones_gui.py`.

Ejecución (PowerShell):

```powershell
python .\\aviones_gui.py
```

Nota: `tkinter` suele estar incluido con Python en Windows. Si no funciona, instala una distribución de Python que incluya `tkinter`.
